\maketitle


Use Claude Code to query the Rice Data Portal for monthly stock price
data (end-of-month \texttt{closeadj} from the SEP table) for five ETFs
of your choice (e.g., SPY, AGG, GLD, VNQ, EFA) and the one-month T-bill
rate from FRED. Have Claude Code compute monthly returns and then ask it
to:

\begin{enumerate}
\def\labelenumi{(\alph{enumi})}
\tightlist
\item
  Estimate expected returns, standard deviations, and correlations from
  the historical data
\item
  Compute the tangency portfolio weights, expected return, standard
  deviation, and Sharpe ratio
\item
  Plot the efficient frontier and the capital allocation line
\item
  Re-solve with a \textbf{no-short-sales} constraint (all weights
  \(\geq 0\))
\item
  Re-solve with a \textbf{maximum 40\% allocation} constraint (no single
  weight \(> 0.40\))
\end{enumerate}

Compare the portfolio weights and Sharpe ratios across all three cases.
Write a one-page memo discussing how constraints affect diversification
and performance.

\bigskip

\noindent\textbf{Deliverables.}

\begin{itemize}
\tightlist
\item
  Data file (\texttt{2A-Data.xlsx})
\item
  Plot showing all three efficient frontiers (\texttt{2A-Plot.png})
\item
  Memo (\texttt{2A-Memo.pdf}, 1 page)
\end{itemize}
